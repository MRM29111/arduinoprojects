Тека "Stepper_speed" містить код для керування кроковим двигуном, під'єднаним до апаратної платформи Arduino. 
Код наведений у файлі двох форматів : .ino (робочий sketch для Arduino) та стандартний текстовий документ.

Схема підключення розділена на дві частини і представлена двома файлами розширення .png.

Для реалізації проекту знадобляться: апаратна платформа arduino, набірне поле, кроковий двигун 28BYJ, кнопки (3 шт.), резистори
опором 1 КОм (3 шт.), провідники (13 шт.)
